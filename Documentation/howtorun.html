How to run the software

Here you will find all the necessary instructions to successfully run this software. Please follow the steps sequentially to avoid unexpected results.

  • The "Code" folder containes all the essential header files and also the main.cpp besides it containes other subfolder as follows:



  • Before you proceed any furthur, please make sure that you followed all the steps How to set up SFML

  • The "Documentation" folder containes all the essential documentation in html files.

  • Now in the subfolder "Code", open main.cpp in your default IDE. Build and Play !!
